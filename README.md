# User Authentication System

A secure User Authentication System built using Python, Flask, and SQLite. This project allows users to register, log in, access protected pages, and log out securely.

## Features

✅ User Registration

✅ User Login

✅ Secure Password Hashing

✅ Session Management

✅ Protected Routes

✅ User Logout

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

## How to Run

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open your browser and visit:

```text
http://127.0.0.1:5000
```
